À propos

PrixDenrées

Cartographie et suivi mensuel des prix des denrées alimentaires de base dans les marchés informels urbains du Cameroun

Objectif

Cette application permet de collecter, analyser et visualiser les prix des denrées alimentaires dans les marchés informels de villes camerounaises (Yaoundé, Douala, Bamenda, Bafoussam, etc.). Elle offre :

  • Collecte de données via formulaire dynamique (saisie manuelle ou import CSV/JSON)
  • Analyse descriptive : statistiques de base, visualisations, tableaux croisés, corrélations
  • Techniques avancées : régression linéaire, ACP, k-means, k-NN, arbre de décision
Justification du thème
Collecte simple
Pas de capteurs coûteux. Un smartphone + formulaire suffisent. Les prix sont publics.
Utilité sociétale
L'inflation alimentaire est un enjeu majeur. Ces données aident ménages et pouvoirs publics.
Manipulable
Données structurées, géolocalisées et temporelles → compatibles Python, SQL, ML.
Guide d'utilisation

  1. Allez sur la page Collecte
  2. Saisie manuelle : Remplissez le formulaire (ville, marché, produit, prix, unité) et cliquez « Enregistrer »
  3. Import fichier : Téléchargez un fichier CSV ou JSON contenant les colonnes requises, prévisualisez-le, puis importez
  4. Vous pouvez aussi charger des données d'exemple depuis la page d'accueil

  1. Assurez-vous d'avoir des données dans la base
  2. Allez sur la page Analyses
  3. Les statistiques descriptives, histogrammes, boxplots, corrélations et évolutions de prix sont calculés automatiquement

  • Régression linéaire : Choisissez une variable cible numérique et des variables explicatives, puis exécutez
  • ACP : Sélectionnez les variables numériques (ou toutes) pour réduire la dimension
  • K-Means : Choisissez le nombre de clusters et les variables pour le clustering
  • K-NN : Sélectionnez une variable cible catégorielle et la valeur de k
  • Arbre de décision : Choisissez une variable cible catégorielle pour la classification
